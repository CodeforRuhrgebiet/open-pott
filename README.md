# open-pott
Wie schaut es mit offenen Daten im Ruhrpott aus?

Wir wollen visualisieren:
* Bochum
* Dortmund
* Duisburg
* Essen
* Gelsenkirchen
* Herne
* Mülheim an der Ruhr
* Oberhausen

(8 größte Städte im Pott)

#### Development

```
npm install
```

**Generate Map data**

Update allowed cities in `generate-map-data.js` and run:

```
node generate-map-data.js
```
